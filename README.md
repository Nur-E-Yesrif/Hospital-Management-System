# 🏥 Hospital Management System

A Hospital Management System built with Java and MySQL.

---

## 📌 About

This project helps hospitals manage patients, doctors, appointments, billing, and medical records efficiently.

---

## ✨ Features

- 👨‍⚕️ Doctor Management
- 🧑‍🤝‍🧑 Patient Management
- 📅 Appointment Booking
- 💊 Pharmacy
- 💵 Billing System
- 📋 Medical Records
- 🔐 Admin Login

---

## 🛠 Technology

- Java
- MySQL
- JDBC
- NetBeans
